    An idiomatic monix service client for Amazon S3.

    An idiomatic monix service client for Amazon S3.

    It is built on top of the software.amazon.awssdk.services.s3, that's the reason why all the methods expects an implicit instance of a S3AsyncClient to be in the scope of the call.

    Each of the methods expects at least the required parameters to build the AWS S3 request plus optionally the most relevant ones that allow advanced settings such as encription and request payer etc. On the other hand all methods but multipart upload accepts the native aws requests to be passed.
